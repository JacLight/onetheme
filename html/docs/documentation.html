Introduction

You bought an HTML / CSS template, so you need some basic knowledge for those 2 languages. The Best way is to play around with the different elements/sections but you find some helpfull tips and explanations how this template is build up and works.
Please compare all the explanation from this file with the original files to have a better understanding.

Introduction

These are general classes, id's or tags which you will find several times in almost all html files.

Every part of website is wrapped up in section tag. Every section tag has predefined margin and custom styling that provides better readability. <section class="gallery" id="gallery">
<div class="container">
...
</div>
</section>
On top of every section you can place title of section. <div class="row headline">
<h1>Our portfolio projects</h1>
<hr />
</div>

RTL text

From version 1.2 ONE is compatible with RTL text.

To activate it, add .rtl class to <body> tag. <body class="home rtl" dir="rtl">
Be aware that with RTL text direction you can only use vertical slider. It will change direction of sliding automatically.

Header

Header is using two parts- one for slider in background, and one displaying navigation and logo.

To create new slide, create new .swiper-slide div inside of .swiper-wrapper. It should look something like that: <div class="swiper-wrapper">
<div class="swiper-slide" style="background-image:url('img/background_Image.png')">...</div>
</div>

Thanks to applying background-image, you can create responsive slide, with parallax effect on image. If you wish to create text, or other content visible on slide, just put it inside of .swiper-slide tag. It would be automatically aligned and formatted to be responsive.

Services

Services section is stylised with services class.

<section class="services">...</section>

This section also uses slider. To insert new slide, use standard Bootstrap markup. Inside of

<div class="carousel-inner"> insert new slides just as you would do with Bootstrap Carousel, <div class="item">

Header

ONE allows placing full width HTLM5 Video. To do so, create <section class="video"> and place <video id="vid1"> <source src="videos/videoSource.mp4" type="video/mp4"> </video>

Header

Call to action in the simplest section of the ONE. You can create on following this markup: <section class="callToAction">
...
</section>
And placing headlines or buttons inside of it.

Gallery

Gallery on the other hand is the most complicated of sections. Create section using <section class="gallery"> Then, using data-path-hover attribute on hover element you can specify the path of hover shape. The ONE comes with triangle as it’s shape, and it’s declared as: <figure data-path-hover="m 0 0 L 100 100 L 100 100 L 0 100 z"> Notice how two of the vertexes shows in one place, that’s because we are morphing rectangle into triangle. If you want to customize starting shape, you can do it using path element. Rectangle code looks like this: <path class="background" d="M 0 75 L 100 75 L 100 100 L 0 100 z"/>
</path> After creating place for the description of image, you can now create one. Use <figcaption>. Gallery also supports touch slider. Create slides just like in header, creating <div class="swiper-slide"> for each of the pages.

Features

Features section is created by placing <section class="features"> in HTML code. To enable scrolling effects add scrollAnimation class to this section.

Blog

<section class="blog"> creates blog section. Use <article> for each of the articles. Place thumbnail inside of <figure>.

Team

Team comes with custom carousel. After creating section with <section class="team">, you can insert new members using <div class="member"> inside of <div class="team-carousel">.

Map

Map is using original Google Maps API. Functions managing it are located inside of Javascript script.js file. You can edit details of infobox inside of <div id="infobox"> in HTML.

Contact

After placing <section class="contact"> inside of your HTML, you can then create input fields using <input type="text" class="form-control" placeholder="Your text">

Voice Control

Adding New Commands

On line 319 of /js/script.js begins function responsible for voice commands. To edit them, simply delete or add new commands object. For example, if you only want two commands, replace lines from 329 to 369 (var commands) with: var commands = {
'services': function() {
goTo("#services");

},

'portfolio': function() {
goTo("#portfolio");

}
};
